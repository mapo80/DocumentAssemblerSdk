# Example 12 - Mail Merge (Edited.docx)

This project validates the `Edited.docx` template copied from `DocumentAssembler/Documents/MailMerge/Edited.docx`.
It ensures the expected MailMerge fields (`FirstName`, `LastName`) are detected.

## Run

```bash
cd DocumentAssembler/DocumentAssemblerSdk.Examples/Example12_MailMerge_Edited
export PATH="$HOME/dotnet:$PATH"
dotnet run
```
