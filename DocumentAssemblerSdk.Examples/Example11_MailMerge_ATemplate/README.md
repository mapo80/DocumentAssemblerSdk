# Example 11 - Mail Merge (ATemplate.docx)

This example loads the `Template.docx` file copied from `DocumentAssembler/Documents/MailMerge/ATemplate.docx`
and verifies that all MailMerge fields (`FirstName`, `LastName`) are detected by the
`TemplateSchemaExtractor`.

## Run

```bash
cd DocumentAssembler/DocumentAssemblerSdk.Examples/Example11_MailMerge_ATemplate
export PATH="$HOME/dotnet:$PATH"
dotnet run
```
